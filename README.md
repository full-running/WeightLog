# 📂 Project Info

## ⚖️ WeightLog

체중 변화를 날짜별로 기록하고, BMI 계산 및 기록 관리를 할 수 있는 웹 애플리케이션입니다.

### 주요 기능

- 날짜 + 체중 입력 및 저장
- 기록 목록 조회 / 수정 / 삭제
- BMI 자동 계산 및 상태 표시
- LocalStorage 기반 데이터 유지 (새로고침 후에도 유지)

### Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
